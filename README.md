# 📌 Customer Support System (Ticket Resolution Tree)

## 📖 Description
This project is a Customer Support Ticket Management System implemented in C using a Binary Search Tree (BST).

It allows CRUD operations on support tickets:
- Create Ticket
- View Tickets
- Search Ticket
- Update Ticket
- Delete Ticket

---

## ⚙️ Features
- Dynamic memory allocation (malloc)
- Tree-based ticket storage
- Efficient searching using BST
- Menu-driven interface

---

## 🧠 Data Structure Used
Binary Search Tree (BST)

Each node contains:
- Ticket ID
- Issue Description
- Status
- Left and Right child pointers
